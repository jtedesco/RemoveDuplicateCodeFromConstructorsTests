final boolean org.eclipse.jdt.ui.tests.refactoring.infra.JavaProjectHelper.ASSERT_NO_MIXED_LINE_DELIMIERS = false [static, private]

If set to true all resources that are deleted using delete(IJavaElement) and that contain mixed line delimiters will result in a test failure.

Should be false during normal and Releng test runs due to performance impact and because the search plug-in gets loaded which results in a test failure.
